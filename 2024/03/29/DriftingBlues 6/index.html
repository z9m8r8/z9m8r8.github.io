1. 信息收集

1.1. 主机探测

1.2. 端口扫描

仅扫描到了 80，还有个 /textpattern/textpattern/ 目录

1.3. Web 信息收集

扫目录

/textpattern/textpattern/ 下是 TextPattern CMS 的登录页面，爆破估计没戏……根据提示继续扫目录

找到个 spammer.zip

解压要密码，用 fcrackzip 爆破下试试

得 mayer:lionheart ，应该是 TextPattern CMS 的登录账号及密码，登录尝试 getshell

2. 渗透攻击

2.1. 获取 shell

看 CMS 主页页脚提示是 v4.8.3，到 Exploit Database - Exploits for Penetration Testers, Researchers, and Ethical Hackers 找找历史漏洞

以 TextPattern CMS 4.8.7 - Remote Command Execution (Authenticated) - PHP webapps Exploit 为例，上传 shell

成功！尝试反弹 shell【Site Unreachable】

在 /etc/passwd 中没看到普通用户，可能需直接提权至 root

2.2. 提权

找到个 config.php

尝试进数据库

找到个 txp_users 表，但仅有 mayer 的信息，用户权限受限 mysql 直接提权未果……

/var 下又找到个 get.zip，解压后仅是个站点的备份……

上传 pspy64 执行也未果……

考虑利用 Linux 内核提权了，看了 提权学习之旅——Linux操作系统提权 - 先知社区 经测靶机存在 Dirty COW，CVE-2016-5195 漏洞

靶机下载 dirty.c 【dirtycow/dirty.c at master · firefart/dirtycow · GitHub】编译

提权

拿到 flag!
