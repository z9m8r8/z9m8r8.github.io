靶机：DriftingBlues: 2 ~ VulnHub

1. 信息收集

1.1. 主机探测

1.2. 端口扫描

21 允许匿名登录，且有个 secret.jpg

ftp 登录下载，并在 kali 分析，啥也没找的……

1.3. Web 信息收集

扫目录

wpscan 扫下 WordPress

枚举出个 albert 用户

看 wp 站点源码猜测可能需要修改 hosts 让 driftingblues.box 指向 192.168.159.132

修改后尝试子域挖掘……没找到，换字典继续尝试目录遍历……也依旧没找到有价值的信息

尝试对 wp 进行 SQL 注入……失败……尝试直接爆破 albert 用户

得 albert / scotland1

也可用 hydra 爆破【Hydra爆破[无验证码]Web页面_hydra 网页-CSDN博客】

注，这里为什么不用 incorrect 作为关键字是因为密码为空时响应并不包含 incorrect ！

2. 渗透攻击

2.1. 获取 shell

登录 wp：albert / scotland1，经测试该用户有修改主题的权限，直接反弹 shell【php | GTFOBins】

kali 先开启监听

2.2. 提权

freddie 下找到了该用户的ssh登录公私钥，利用 scp 将私钥传到 kali 尝试破解

经测没有密码

ssh 直接登录 freddie 用户，基于 freddie 用户进一步尝试提权

flag 1

nmap 提权【nmap | GTFOBins】

拿到 flag 2！
