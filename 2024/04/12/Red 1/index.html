2.2. 提权

操作时发现，shell 会被不断强制下线，密码也会不断被修改！应该有定时任务，可重新爆破获取

sudo -l 找到个 /usr/bin/time ，可利用其提权至 ippsec 用户【time | GTFOBins】，不过依旧会被定时强制下线

直接反弹个 shell，避免被定时强制下线

ssh 登录 john 用户

反弹的 shell 中切换用户

获取完全的 shell

上传 pspy64 执行看看

经测 cat 和 vi 的功能互换了，用 vi 读取 supersecretfileuc.c【亦可尝试 tac,more 命令】

supersecretfileuc.c 会定时编译执行，查权限

supersecretfileuc.c 编译的 rev 有 root 权限，替换 supersecretfileuc.c 可以反弹个 root 用户的 shell【反弹shell生成器】

kali 开启监听等待即可！
