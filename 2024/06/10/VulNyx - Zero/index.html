2.1. 获取 shell

利用 RCE 反弹 shell

2.2. 提权至 liam

在 .bash_history 中读到 liam 的密码 L14mD0ck3Rp0w4 ，ssh 登录 liam

2.3. 提权至 root

liam 用户 sudo -l 得 (root) NOPASSWD: /usr/bin/wine

Wine 是一个允许你在类Unix操作系统（如 Linux）上运行 Windows 应用程序。Wine 通过对 Windows 应用程序的二进制文件进行逆向工程和模拟，让这些应用程序能够在没有安装 Microsoft Windows 的情况下运行。

利用 msfvenom 生成个反弹 shell【试了几个 windows 的 payload 都没成功，但下面 linux 的 payload 可成功】

将 reshell.exe 传至靶机利用 wine 执行

获得 root 权限

也可直接用 wine 直接进入 cmd ，但操作有诸多不便，不推荐
