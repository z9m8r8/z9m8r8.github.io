2. 渗透攻击

2.1. 获取 shell

Tomcat 渗透：Tomcat | HackTricks | HackTricks

上传反弹 shell

得 tomcat 用户 shell

2.2. 提权至 sa

tomcat-users.xml 中读到个 sa 用户密码，而经测系统亦存在该用户

尝试 su sa 并未成功，但经测 ssh 可以直接登录

获得 sa 用户权限

2.3. 提权至 toor

上传 pspy64 执行得系统存在定时任务

往 /var/www/html 下传个 shell 试试

经测反弹得到的是 toor 用户的 shell

补充：ps aux |grep "toor" 可得 apache2 是 toor 启动的

2.4. 提权至 root

sudo -l 得 (root) NOPASSWD: /usr/bin/ex

ex 提权：ex | GTFOBins

为方便利用 ex 提权，先利用 ssh-keygen 为当前用户生成个 ssh 私钥，kali 再 ssh 登录 toor 来提权

获得 root 权限
