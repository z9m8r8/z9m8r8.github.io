靶机：Grotesque: 3.0.1

1. 信息收集

1.1. 主机探测

靶机：192.168.159.167

1.2. 端口扫描

1.3. Web 信息收集

根据 80 的提示分析 atlasg.jpg

尝试对图片隐写分析

没找到有用信息……

尝试扫目录

依旧没扫到什么，再看看图片

提示 md5，尝试将目录字典转为相应 md5 值后再扫描

尝试文件包含，Fuzzing

得参数 purpose

扫下 linux 敏感文件

/etc/passwd 中找到个 freddie 用户，没找到其他有用信息，经测 php 伪协议也写不了 shell ，尝试爆破 freddie

rockyou.txt 没跑出来，但借鉴扫目录的操作，将 rockyou.txt 转为 MD5 后爆破得到了 freddie/61a4e3e60c063d1e472dd780f64e6cad

2. 渗透攻击

2.1. 获取 shell

ssh 登录 freddie/61a4e3e60c063d1e472dd780f64e6cad

2.2. 提权

上传 pspy64 监测到个定时任务 /bin/sh -c bash /smbshare/*

可以考虑往 smbshare 下写个提权 shell ，不过 freddie 用户在 smbshare 下没写权限

经测 freddie 用户利用 smbclient 连接后可成功上传个 shell
