2. 渗透攻击

2.1. 获取 shell

经测存在命令注入漏洞

读取源码看看

pingit.php

传入的参数未作任何过滤，拼接后利用 shell_exec() 函数执行了

kali 开监听，反弹 shell

获得 apache 权限 shell

2.2. 提权

经测系统内核存在 CVE-2009-2698 漏洞，可用来提权

获得 root 权限

3. 复盘

3.1. SQL 注入

post.txt

index.php
