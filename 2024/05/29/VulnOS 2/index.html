1.3. Web 信息收集

80 主页 \(\Rightarrow\) jabc \(\Rightarrow\) Drupal 7，扫目录并找历史漏洞

robots.txt 提示的 URL 大多数访问都是 404 ，没找到有用信息

searchsploit drupal 查询相关漏洞，经测 msf 中 unix/webapp/drupal_drupalgeddon2 可成功利用

2. 渗透攻击

2.1. 获取 shell

利用 msf 获得了个 www-data 的 shell ，基于此拿个完全的 shell 以便提权

msf shell

kali 命令

2.2. 提权

查询 suid

有 polkit ，尝试 CVE-2021-4034 提权

提权成功，拿到 flag

3. 回顾

3.1. OpenDocMan

jabc Documentation 页面源码有提示 /jabcd0cs/

访问是部署的 OpenDocMan v1.2.7 ，可用 guest/guest 登录

searchsploit OpenDocMan 得当前版本存在 SQL 注入漏洞

/usr/share/exploitdb/exploits/php/webapps/32075.txt

跑 sqlmap

得 vulnosadmin 和 webmin 用户名

jabcd0cs 数据库读到用户信息

b78aae356709f8c31118ea613980954b \(\Rightarrow\) webmin1980【MD5免费在线解密破解-SOMD5 】

系统亦存在 webmin 用户，经测 ssh 可直接登录

注，webmin 用户登录 OpenDocMan 后可上传文件，不过后端上传的文件会被重命名为 x.dat ，没法利用
